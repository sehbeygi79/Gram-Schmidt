# Gram-Schmidt

This is a simple implementation of Gram-Schmidt algorithm.
The code gets n independent vectors and returns the corresponding orthonormal vectors in ascending/descending order.

sample input:
```
3 3
0 3 4
1 0 1
1 1 3
```
sample output
```
0.857 -0.412 0.309 
0.000 0.600 0.800 
-0.514 -0.686 0.514
```
